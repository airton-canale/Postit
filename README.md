## Installation

- Important: You must have installed the Next.js and Git

```bash
# clone it
$ git clone git@github.com:airton-canale/postit.git

$ cd postit

# Install dependencies
$ npm install

# Running scripts
$ npm run dev
```
- And enter in your browser using http://localhost:3000
